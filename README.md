# EmployeeManagement

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 21.1.0.

## Environment Requirements

Before running this application, ensure you have the following installed:

- **Node.js**: v18.x or higher
- **npm**: v9.x or higher (comes with Node.js)
- **Angular CLI**: v21.1.0 (will be installed as dev dependency)

### Check Your Environment

```bash
# Check Node.js version
node --version

# Check npm version
npm --version
```

## Installation

1. **Navigate to the project directory:**
   ```bash
   cd "Test Assessment - FE Angular - Jeffry Alexander"
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

   This will install all required dependencies including Angular CLI and Angular Material.

## Running the Application

### Development Server

Run the development server:

```bash
npm start
```

or

```bash
ng serve
```

The application will be available at `http://localhost:4200/`

The application will automatically reload if you change any of the source files.

### Production Build

To build the project for production:

```bash
npm run build
```

The build artifacts will be stored in the `dist/` directory.

### Watch Mode

To build and watch for changes:

```bash
npm run watch
```
